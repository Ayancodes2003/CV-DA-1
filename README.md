# Interactive Shape & Contour Analysis (restored)

This project implements a contour-based shape detection system using OpenCV. The dashboard is implemented with Streamlit. The repository contains code for detecting shapes, computing area and perimeter, and an interactive UI for demonstration and grading.

Run instructions:
1. python -m venv .venv
2. .\.venv\Scripts\activate
3. pip install -r requirements.txt
4. streamlit run app.py
